Turn off Wi-Fi and keep writing. Nothing changes — that's the guarantee, not a setting you can forget to switch on. For a rigorous proof, run privacy/no_egress_test.sh.

WHY THIS MATTERS

Your unpublished drafts are sensitive intellectual property. A cloud writing assistant may log, retain, or train on what you send it. Bardbox never makes that trade: the model is a file on your disk, and the app has no network code path to anywhere but your own machine.
